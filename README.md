# Lede

Claude Code plugin: see what you could have said in your first prompt.

`/assess` analyzes the chunk of session ending at the moment you typed it, and emits a diff between the original first prompt and a stronger starting point for similar future work — labeled foldable (could have been said upfront) or emergent (only knowable by trying).

Implementation in progress. See `docs/superpowers/specs/2026-05-09-lede-design.md` for design.
